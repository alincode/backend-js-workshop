初始化

初始化專案

語法

npm init [-f|--force|-y|--yes]
  • 如果你加了 -y 或 -f 參數，代表你將認同使用預設的設定值來產生 package.json 檔。
  • init | nam Documentation

範例

npm init
npm init -y
npm init -f

介紹 package.json 檔案
