Node.JS 介紹

Node.JS

什麼是 Node？

https://zh.wikipedia.org/wiki/Node.js

觀念題

  • ❌ 我寫 Node.js
  • ✔️ Node.js 是一個伺服器端的 Javascript 運行環境
  • ✔️ Node.js 是非阻塞的
  • ✔️ Node.js 是事件驅動的，開發者可以在不使用執行緒的情況下開發出一個能夠承載「高並行」的伺服器。
  • ✔️ Node.js 允許通過 JavaScript 和一系列模組來撰寫後端應用和網路相關的應用。
