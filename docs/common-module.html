常用套件

常用模組

  • moment.js: 與日期與時間相關的常用模組
  • lodash: 常見實用函示庫
  • intl-messageformat：處理多國語系
  • mongoose: 優雅的操作 MongoDB
  • Sequelize - Multi-dialect ORM. 支持 PostgreSQL, SQLite, MySQL.
  • jsonwebtoken JWT 相關模組
  • joi: validator 模組
  • express: Web application framework
  • koa: 作者是設計 express 框架的同班人馬，但提供更彈性級的架構。
  • node-fetch: API 介面與 window.fetch 一致，可以執行在 Node.js 環境。
