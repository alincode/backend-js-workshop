- BDD 是一種從外到內的方法，實際上是針對業務驅動程序的。
- 行為驅動開發 (BDD) 是一種敏捷軟體開發的技術，它鼓勵軟體項目中的開發者、 QA 和非技術人員或商業參與者之間的協作。
